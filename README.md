# teste_dev_wabtec API (Teste Técnico)

Este projeto contém a estrutura base de uma API em C#.

## Objetivo do teste
Implementar endpoints:
- GET /tasks
- GET /tasks?isCompleted=true|false
- POST /tasks

## Orientações
Utilize apenas armazenamento em memória.
Foque em clareza, organização.
Boas práticas e arquitetura limpa serão um diferencial na avaliação.
Não é necessário banco de dados, autenticação ou testes.
Configurar rotas e controladores para as operações solicitadas.

## Orientações de entrega
1. Clone este repositório.
2. Implemente as funcionalidades solicitadas.
3. Faça um commit com suas alterações.
4. Envie um pull request para este repositório.
5. Inclua uma breve descrição das mudanças feitas no pull request.
6. Aguarde a revisão do código e feedback.
